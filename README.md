# Aaryesh Shukla - Interactive Resume

## Overview
This is a premium interactive resume for Aaryesh Shukla, designed to showcase education, skills, projects, and contact information in a visually stunning manner.

## Features
- Glassmorphic dark theme with smooth transitions
- Sticky navigation bar with active section indicators
- Interactive project cards with modal details
- Skills summary with progress bars
- Contact form with real-time validation

## Technologies Used
- HTML
- CSS
- JavaScript
- Google Fonts (Outfit, Inter)

## Setup Instructions
1. Clone the repository or download the files.
2. Open `index.html` in your web browser.
3. Enjoy the interactive resume!
